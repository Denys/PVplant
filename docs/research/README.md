# Research

Store project research notes here. Separate confirmed facts, assumptions, recommendations, and open risks.

Use `pvplant-iwin-research-template.md` for new notes.

Preferred source order for iWin/BIPV research:

1. Root `AGENTS.md`, `CHECKPOINT.md`, `PROJECT_TRACKER.md`, and `.memory/`.
2. Companion files under `BIPV_Codex_Edition/companion/`.
3. Current Daily reports and source inventories.
4. NotebookLM source-pack manifest and required source folders.
5. Official datasheets, standards, manuals, and papers.
6. Presentation decks and public-clue files only as project-claim context unless corroborated.

Every research note should state whether it changes the current A4/A1/A2/A3 ranking, and what evidence would overturn it.
