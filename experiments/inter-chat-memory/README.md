# Inter-Chat Memory Experiment

Use this folder for experiments that improve durable project memory across agent sessions.

Acceptable experiments:

- source-card formats for faster iWin/BIPV evidence recovery;
- compact state mirrors for split-chat continuation;
- tests for whether future agents can recover A4/A1/A2/A3 state without chat history;
- checks that vendor-data-required items do not get rewritten as facts;
- comparison of `.memory/` hub layouts against NotebookLM source-pack load order.

Do not store final project truth here. Promote useful outcomes into `.memory/`, `CHECKPOINT.md`, `PROJECT_TRACKER.md`, or `docs/research/`.
